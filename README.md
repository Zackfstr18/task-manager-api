# Task Manager API

Esta API permite crear y administrar tareas personales.

## ¿Qué problema resuelve?
Permite guardar tareas y consultarlas desde cualquier aplicación.

## Funciones principales
- Listar todas las tareas
- Ver una tarea por ID
- Crear una tarea
- Marcar una tarea como completada
- Eliminar una tarea

## Tecnologías
- C#
- ASP.NET Core Web API
- .NET 8
